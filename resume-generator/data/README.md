# 配置文件说明

## 📁 data目录配置文件

### 1. sample_config.xlsx
**用途**: 基础简历生成配置示例
**适用场景**: 快速体验工具功能
**包含内容**: 
- 完整的合同、角色、人员配置示例
- 可直接用于测试简历生成功能

### 2. contract_role_mapping.xlsx  
**用途**: 合同-角色映射配置模板
**适用场景**: 基于实际合同文件进行配置
**包含内容**:
- 合同基本信息
- 角色需求定义
- 技能和经验要求
- 人员数量配置

### 3. personnel_template_example.xlsx
**用途**: 人员配置模板示例
**适用场景**: 参考人员信息填写格式
**包含内容**:
- 人员基本信息字段说明
- 技能和经验填写示例
- 配置格式参考

## 🚀 使用建议

### 快速开始
使用 `sample_config.xlsx` 直接测试工具功能

### 正式使用
1. 运行 `python3 contract_mapper.py` 分析合同
2. 配置 `contract_role_mapping.xlsx` 中的角色需求
3. 工具自动生成详细配置文件
4. 在生成的配置中填写具体人员信息

## 💡 注意事项

- 配置文件为Excel格式，建议使用Microsoft Excel或WPS Office编辑
- 确保字段格式正确，特别是日期和数字格式
- 建议在修改前备份原始配置文件
